# Security Policy

## Supported Versions

We currently support the latest releases for active security patching:

| Version | Supported |
| ------- | --------- |
| 1.1.x   | Yes       |
| 1.0.x   | No        |

## Reporting a Vulnerability

Please do not open GitHub Issues for security vulnerabilities. Instead, report security bugs privately to the repository maintainers. 

We will acknowledge your report within 48 hours and work with you to resolve the concern immediately.
