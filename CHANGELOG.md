# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-06-27
### Added
- **SaaS settings manager**: Persists LLM Providers, endpoints, and themes configs.
- **Engineering Metrics dashboard**: Tracks memory, process latencies, cache rates, and tool volumes.
- **Smart cache manager**: Dynamic cache expiration logic.
- **Bundled ZIP exporter**: Assembles full report bundle (CSV, Excel, PDF, PPTX, JSON, Markdown).
- **One-click Demo mode**: Generates transactional financial sales dataset.
- **Connection Lost banners**: Alerts users of backend disconnects.
- **Docker compositions**: Configured backend and frontend Dockerfiles.
- **CI/CD Actions workflow**: Automates compile builds and pytest validations.
- **GitHub templates**: Added issue templates, PR template, CODE_OF_CONDUCT.md, LICENSE, and SECURITY.md.

## [1.0.0] - 2026-06-15
### Added
- Professional React user dashboard layout.
- Ingestion engine with health checking grading.
- LangGraph agent planner and executor nodes.
- PDF and PowerPoint slide document services.
